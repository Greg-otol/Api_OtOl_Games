# Module 5 - Project 2

* In this project, the technologies NestJS and Prisma were used.

## After downloading or cloning the project, install the dependencies:

Assuming that NodeJS is installed on the machine, run the command:

### `npm install`

## Settings that must be done before running:

* Create a `.env` file in the project root and paste the variables that are in `.env.example`

* Edit the password, port and database name in the `DATABASE_URL` variable with your information

* Open terminal and run `npx prisma migrate dev` command to create Database and tables

## To run the project locally:

### `npm run start`

### `open local server http://localhost:3001`
